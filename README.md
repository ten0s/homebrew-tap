[Homebrew Tap](https://docs.brew.sh/Taps) for
[VeLisp](https://github.com/ten0s/velisp)

## Install

```
% brew tap ten0s/velisp
% brew update
```

```
% brew install libgimacos
% brew install slide
```

## Uninstall

```
% brew uninstall libgimacos
% brew uninstall slide
```

```
% brew untap ten0s/velisp
```
